# ICF ACC Coaching Evaluation Tool

A lightweight tool to evaluate coaching sessions based on ICF Core Competencies 2025.

## Features
- Structured competency scoring (1–5)
- Automated average scoring
- Pass / Borderline / Fail output

## Usage
1. Open index.html
2. Score each competency
3. Click Submit

## Future Enhancements
- Save results to database
- Multi-user login
- Dashboard analytics
